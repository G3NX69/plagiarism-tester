Setup Instructions

1

Create Repository

Create a new repository on GitHub:

  • Name: plagiarism-detector
  • Visibility: Public
  • Initialize with a README
Create Repository
2

Clone Repository

Clone your repository to your local machine:

$ git clone https://github.com/G3NX09/plagiarism-detector.git

Or using SSH:

$ git clone git@github.com:G3NX09/plagiarism-detector.git
3

Add Files

Create your project files in the repository folder:

$ cd plagiarism-detector
$ echo "# Plagiarism Detector" >> README.md
$ touch index.html styles.css script.js

Add your plagiarism detector code to these files.

4

Commit Changes

Commit your files to the repository:

$ git add .
$ git commit -m "Initial commit with plagiarism detector"
$ git push origin main
5

Enable GitHub Pages

Set up GitHub Pages for your repository:

  1. Go to repository Settings
  2. Select "Pages" from the left menu
  3. Set source to "main" branch
  4. Save and wait for deployment

Your site will be available at:

https://G3NX09.github.io/plagiarism-detector

Command Line Setup

For quick setup from an existing directory:

$ echo "# plagiarism-detector" >> README.md
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git branch -M main
$ git remote add origin https://github.com/G3NX09/plagiarism-detector.git
$ git push -u origin main
